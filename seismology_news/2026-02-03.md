# 地震學與地球物理研究每日摘要 (2026-02-03)

## 1. 自監督擴散框架用於海洋地震數據重建 (arXiv:2602.01909)
**標題**: Propagating the prior from far to near offset: A self-supervised diffusion framework for progressively recovering near-offsets of towed-streamer data

### 內容摘要
在海洋拖曳式電纜地震採集（towed-streamer acquisition）中，由於最靠近震源的檢波器（hydrophone）通常距離震源兩百公尺以上，導致「近偏移距（near-offset）」數據缺失。這會嚴重影響後續的地震數據處理程序，如表面相關多次波消除（SRME）、速度分析和全波形反演（FWI）。
本研究提出一種基於**自監督擴散模型（self-supervised diffusion-based framework）**的框架，利用現有的遠偏移距數據來訓練模型，學習偏移距相關的統計特徵（如事件曲率、振幅變化和子波特性），並透過遞歸外推技術精準重建缺失的近偏移距追蹤。

### 專業解說：地震學意義
- **地震成像精度提升**：近偏移距數據對於淺層構造的成像至關重要。透過精準重建，能大幅提升地殼淺層（如斷層帶、沉積層）的成像解析度，有助於更準確地理解區域地殼構造。
- **災害減輕與風險評估**：高品質的海底構造影像是評估海底地震引發海嘯風險的基礎。此技術能減少野外採集受限帶來的數據誤差，為災害模擬提供更可靠的地層參數（如 P 波、S 波速度）。

## 2. 機器學習確定 Mg2SiO4 的 P-T 相圖 (arXiv:2602.01730)
**標題**: Machine learning determines the Mg2SiO4 P-T phase diagram

### 內容摘要
Mg2SiO4（橄欖石及其高壓同質異形體：瓦茲利石 wadsleyite 和 林伍德石 ringwoodite）的相變是地函動力學和深部物質循環的核心。然而，在高溫高壓極端環境下，實驗室確定的相邊界仍存在爭議。
本研究結合了**機器學習勢能（Machine Learning Potential, MLP）**、非平衡熱力學積分（NETI）與兩相共存模擬，構建了完整的壓力和溫度（P-T）相圖。

### 專業解說：地震學意義
- **地殼與地函構造理解**：橄欖石相變發生在 410 公里與 660 公里不連續面。此研究提供了更精確的相圖，有助於解釋地震波在過渡帶（Transition Zone）中的速度跳躍，進而推論地函的溫度結構與組成。
- **地函動力學模擬**：準確的相圖是模擬地函對流、板塊俯衝深度及深部地震發生機制的關鍵參數，對於理解全球板塊運動具有重要意義。

## 3. SAIPy 在火山地震監測中的應用評估 (arXiv:2602.01269)
**標題**: Evaluating the SAIPy Performance using a Local Seismic Network for Volcano-Tectonic Earthquakes Monitoring

### 內容摘要
SAIPy 是一個基於 Python 的開源深度學習地震數據分析工具包。本研究評估了其在複雜的火山環境（Caldera）下的表現。雖然 SAIPy 最初是用構造地震數據訓練的，但本研究測試了它在**火山構造地震（Volcano-Tectonic, VT）**監測中的有效性，包括地震偵測、震級估計、相位拾取（picking）和 P 波極性分類。

### 專業解說：地震學意義
- **地震預警與監測自動化**：研究結果顯示 SAIPy 能夠識別出比現有目錄更多的局部地震事件。這證明了 AI 工具在即時處理大量連續監測數據中的潛力，特別是在自動化監測活動火山的微震（Microseismicity）方面。
- **火山災害減輕**：精準的相位拾取和事件定位對於追蹤岩漿侵入路徑至關重要。透過提升自動化處理的精度，監測人員能更快掌握火山活動趨勢，為疏散決策提供科學依據。

## 4. 基於 CPA 的區域土壤水力導度估計 (arXiv:2602.00893)
**標題**: Regional-Scale Estimation of Soil Hydraulic Conductivity Using the Kansas Mesonet

### 內容摘要
雖然主要屬於土壤物理，但對於理解淺層地震效應及場址反應（Site Response）亦具參考價值。研究應用臨界路徑分析（Critical Path Analysis, CPA）從土壤水分特徵曲線估計飽和水力導度（K_sat）。

### 專業解說：地震學意義
- **場址反應與災害評估**：土壤的含水狀態與孔隙構造直接影響地震波（特別是剪力波）在近地表的放大效應。水力導度的研究有助於建立更精細的近地表飽和度模型，這對於評估地震誘發的液化風險（Liquefaction）具有輔助參考價值。
