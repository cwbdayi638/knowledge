應力與應變專題研究 - 系列 3/5：三維彈性常數與拉梅參數

在地震學與地球物理學中，描述岩石如何響應外力（應力）並產生形變（應變）是理解地震波傳播的基礎。本篇專題將深入探討三維空間中的彈性常數，以及在理論推導中極為重要的拉梅參數。

1. 核心彈性常數定義

(1) 帕松比 (Poisson's Ratio, ν)
- 定義：當材料在一方向受壓或受拉時，其橫向應變與縱向應變之比的負值。
- 公式：ν = -ε_trans / ε_axial
- 物理意義：反映材料的側向變形能力。對於大多數岩石，ν 值約在 0.25 左右；流體的 ν 為 0.5（不可壓縮）。

(2) 體積模數 (Bulk Modulus, K)
- 定義：物體在受到均勻壓力時，其體積應變與所受應力之比。
- 公式：K = -P / (ΔV/V)
- 物理意義：描述材料抵抗「體積改變」的能力，即材料的不可壓縮性。

(3) 剪切模數 (Shear Modulus, μ 或 G)
- 定義：剪應力與剪應變之比。
- 公式：μ = τ / γ
- 物理意義：描述材料抵抗「形狀改變」的能力。流體的 μ 為 0（無法抵抗剪切）。

2. 拉梅參數 (Lamé Parameters: λ, μ)

在等向性線性彈性理論中，應力與應變的關係可以用兩個獨立參數來完整描述，即拉梅參數：

- 第一參數 (λ)：描述材料受正向應力時，對垂直方向變形的貢獻。它沒有直接簡單的物理直觀，但在數學推導中極其便利。
- 第二參數 (μ)：即上述的「剪切模數」，描述材料對剪切形變的抵抗力。

3. 參數間的相互關係

所有彈性常數（E, ν, K, μ, λ）之間都是相互關聯的，已知其中兩個即可求出其他參數。例如：
- K = λ + (2/3)μ
- E = μ(3λ + 2μ) / (λ + μ)
- ν = λ / [2(λ + μ)]

4. 拉梅參數在地震波速中的物理意義

地震波的傳播速度由介質的彈性參數與密度 (ρ) 決定：

(1) P 波波速 (Vp)：
Vp = √[(λ + 2μ) / ρ] = √[(K + 4/3μ) / ρ]
- 物理意義：P 波是壓力波（縱波），其波速取決於材料抵抗體積改變 (K) 與抵抗剪切形變 (μ) 的綜合能力。

(2) S 波波速 (Vs)：
Vs = √(μ / ρ)
- 物理意義：S 波是剪力波（橫波），其波速僅取決於材料抵抗剪切形變 (μ) 的能力。由於流體的 μ = 0，因此 S 波無法在流體（如外核）中傳播。

總結來說，λ 與 μ 簡化了波動方程式的係數，使得地震波速與介質物理性質之間的關係一目了然。
